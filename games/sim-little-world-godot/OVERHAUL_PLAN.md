# Little World: Genesis — 20-part overhaul

This upgrade turns the prototype from a passive ecosystem visualizer into a readable, goal-driven planet stewardship game.

1. Add a cinematic mission briefing before the simulation begins.
2. Give the player a clear five-chapter campaign with visible objectives.
3. Add unlockable life forms and tools so the first minute is understandable.
4. Add Catalyst as a regenerating action resource to make choices meaningful.
5. Add a planet stability score that explains whether the food web is healthy.
6. Add milestone rewards and clear success feedback.
7. Add population caps and species caps to stop runaway entity explosions.
8. Add density pressure to reproduction so populations self-regulate.
9. Reduce the simulation grid and update terrain ecology in staggered batches.
10. Cap render cadence independently from simulation cadence.
11. Replace the noisy “pile of circles” terrain with cleaner biome tiles and coast contrast.
12. Redesign organisms as larger, distinct silhouettes with restrained glow.
13. Reduce ambient particles and weather effects adaptively when the world is busy.
14. Add a Reduced Motion mode for calmer visuals and slower ambient animation.
15. Add simulation speed controls and dependable pause/play behavior.
16. Add keyboard shortcuts for tools, pause, speed, new world, and motion mode.
17. Add brush throttling, a world cursor, action costs, and immediate action feedback.
18. Redesign the HUD around mission, planet health, Catalyst, events, and readable statistics.
19. Improve the population graph with a legend and stable scaling.
20. Move the game to a true 16:9 canvas and rebuild the web export for consistent browser sizing.

## Campaign chapters

- **Awaken the Shallows:** establish enough cyanobacteria to support a food web.
- **Feed the Drifters:** introduce amoeboids and grazers without exhausting the mats.
- **Complete the Web:** introduce predators while keeping stability above collapse.
- **Change the Sky:** grow oxygen and biodiversity through sustained balance.
- **Living Planet:** reach a mature generation and a resilient ecosystem.

The game remains replayable through named seeds and the New World button, but the first session now teaches itself through the chapter objectives.
